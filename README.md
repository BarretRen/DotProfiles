## Introduction

个人工作总结的Linux/Windows上软件配置备份

## NeoVim profile Installation

### Linux

* clone repo
* 将nvim复制到`~/.config`
* open nvim, waiting for plugins installed

### Windosw

* clone repo
* 将nvim复制到`~/AppData/Local`
* open nvim, waiting for plugins installed

## tmux profile installation

* clone repo
* 将tmux复制到`~/.config`
