## Introduction

个人工作总结的Linux上软件配置，基本满足工作编程需要，包含如下软件的配置
* NeoVim
* zsh
* bash
* tmux
* tmx: 自动创建tmux的脚本
* gitconfig

## NeoVim profile Installation

### Linux
```bash
git clone https://github.com/BarretRen/LinuxEnvProfiles.git
cd LinuxEnvProfiles
cp nvim ~/.config
```
### Windosw
* clone repo
* 将nvim复制到`~/AppData/Local`
* open nvim, waiting for plugins installed
